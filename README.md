Proceso de Instalación

Configuración Base de Datos clase -> (database.js)

    host: '127.0.0.1',                //Host DB

    user: '*****',                 //USER DB
    password: '*****'                 //PASSWORD DB 

Iniciar Servicio

  # npm start
